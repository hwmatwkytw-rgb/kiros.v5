# 🤖 N-Dora Bot v2.0

**بوت فيسبوك ماسنجر متقدم مع ميزات ذكاء اصطناعي، وسائط، وخدمات متنوعة**

---

## ✨ الميزات الرئيسية

### 🧠 الذكاء الاصطناعي (AI)
- **ذكاء**: محادثة ذكية مع GPT-4
- دعم اللغة العربية الكامل
- إجابات سريعة وموثوقة

### 🎬 الوسائط (Media)
- **فيديو**: تنزيل من YouTube بأفضل جودة
- دعم تنسيقات متعددة
- معالجة سريعة وآمنة

### 🛠️ الخدمات (Utility)
- **طقس**: معلومات الطقس الحالية والتنبؤات
- **ترجمة**: ترجمة بين 15 لغة
- **ويكي**: البحث على ويكيبيديا
- **نكتة**: نكات عشوائية مضحكة
- **اقتباس**: اقتباسات ملهمة
- **شخص**: بيانات شخص عشوائي

### 👑 أوامر المطور (Admin)
- **مطور**: إدارة البوت والإحصائيات
- إعادة تشغيل البوت
- عرض الإحصائيات
- مسح الذاكرة المؤقتة

---

## 📋 المتطلبات

- **Node.js**: v16.0.0 أو أحدث
- **npm**: v8.0.0 أو أحدث
- **حساب Facebook**: للمصادقة
- **API Keys**:
  - OpenAI API Key (للذكاء الاصطناعي)
  - OpenWeatherMap API Key (للطقس)

## 🚀 التثبيت والتشغيل

### 1. استنساخ المستودع
```bash
git clone https://github.com/12mom/N-Dora.git
cd N-Dora
```

### 2. تثبيت التبعيات
```bash
npm install
```

### 3. إعداد المتغيرات البيئية
```bash
cp .env.example .env
```

ثم عدّل ملف `.env` وأضف مفاتيح API الخاصة بك:
```env
OPENAI_API_KEY=your_openai_key
OPEN_WEATHER_API=your_weather_key
APPSTATE_PATH=appstate.json
```

### 4. المصادقة
```bash
npm run login
```

### 5. تشغيل البوت
```bash
npm start
```

## 📚 قائمة الأوامر

### أوامر الذكاء الاصطناعي
| الأمر | الوصف | الاستخدام |
|------|-------|---------|
| `.ذكاء` | محادثة ذكية مع AI | `.ذكاء ما هو الذكاء الاصطناعي؟` |

### أوامر الوسائط
| الأمر | الوصف | الاستخدام |
|------|-------|---------|
| `.فيديو` | تنزيل من YouTube | `.فيديو [رابط YouTube]` |

### أوامر الخدمات
| الأمر | الوصف | الاستخدام |
|------|-------|---------|
| `.طقس` | معلومات الطقس | `.طقس القاهرة` |
| `.ترجمة` | ترجمة النصوص | `.ترجمة en مرحبا` |
| `.ويكي` | بحث ويكيبيديا | `.ويكي الذكاء الاصطناعي` |
| `.نكتة` | نكتة عشوائية | `.نكتة` |
| `.اقتباس` | اقتباس ملهم | `.اقتباس` |
| `.شخص` | شخص عشوائي | `.شخص` |

### أوامر المطور
| الأمر | الوصف | الاستخدام |
|------|-------|---------|
| `.مطور إحصائيات` | إحصائيات البوت | `.مطور إحصائيات` |
| `.مطور أوامر` | قائمة الأوامر | `.مطور أوامر` |
| `.مطور حالة` | حالة البوت | `.مطور حالة` |

### أوامر النظام
| الأمر | الوصف | الاستخدام |
|------|-------|---------|
| `.اوامر` | قائمة الأوامر | `.اوامر` |
| `.اوامر [اسم الأمر]` | معلومات الأمر | `.اوامر ذكاء` |

## 📝 المتغيرات البيئية

| المتغير | الوصف | مثال |
|--------|-------|------|
| `OPENAI_API_KEY` | مفتاح OpenAI | `sk-...` |
| `OPEN_WEATHER_API` | مفتاح الطقس | `c4ef85b93982...` |
| `APPSTATE_PATH` | مسار ملف appstate | `appstate.json` |
| `PREFIX` | بادئة الأوامر | `.` |
| `BOTNAME` | اسم البوت | `N-Dora Bot` |
| `PORT` | منفذ الخادم | `7986` |

## 🐛 حل المشاكل الشائعة

### المشكلة: "API Key غير صحيح"
**الحل**: تأكد من نسخ المفتاح بشكل صحيح في ملف `.env`

### المشكلة: "خطأ في الاتصال بـ Facebook"
**الحل**: أعد تشغيل المصادقة باستخدام `npm run login`

### المشكلة: "البوت لا يستجيب للأوامر"
**الحل**: تأكد من استخدام البادئة الصحيحة (افتراضياً `.`)

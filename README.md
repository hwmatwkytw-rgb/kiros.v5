## 🎨 الهوية البصرية والواجهة (Visual Identity)
يتميز **KIROS BOT** بواجهة Terminal فريدة تعتمد على:
* **Dynamic Gradients:** تدرجات لونية تتغير حسب حالة البوت (تشغيل، خطأ، تحديث).
* **Glassmorphism Style:** تصميم رسائل البوت داخل المسنجر بتنسيقات بصرية منظمة.
* **Startup Animation:** شاشة ترحيبية متحركة تعرض شعار الكود ومعلومات المطور فور التشغيل.
